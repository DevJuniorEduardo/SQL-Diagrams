Diagramas de Bases de Datos

Este repositorio contiene modelos y diagramas de bases de datos relacionales desarrollados para diferentes casos de uso. Cada carpeta representa un proyecto con su respectiva estructura y documentación.

Estructura del repositorio

```
main/
├── README.md
├── folder/
│   ├── diagram (Diagrama)
│   └── preview.txt (Descripción de las tablas)
```

- `folder/`: Contiene el diagrama o script SQL de la base de datos que se desea compartir.
  - `diagram`: Archivo con el diseño de la base de datos (imagen, SQL o pseudocódigo estructurado).
  - `preview.txt`: Descripción breve de cada tabla, sus relaciones y el propósito del sistema.

Cada base de datos está diseñada para mantener la integridad referencial y facilitar su implementación en un sistema de gestión… o al menos, esa es la idea.

Objetivo

Este repositorio tiene como objetivo practicar el diseño de bases de datos relacionales, comprender las relaciones entre tablas, el uso de claves primarias/foráneas y facilitar su implementación en sistemas backend.
